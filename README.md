# Summer Trip

Road trip planning materials for a Fort Collins, CO to Eugene, OR summer itinerary from May 15 to June 22, 2026.

## Contents

- `itinerary.txt` - day-by-day trip plan with drive estimates, camping notes, river days, backpacking blocks, and permit reminders.
- `trip_stops.csv` - structured stop data with dates, categories, notes, and coordinates.
- `google_maps_route_links.txt` - route links for major driving segments.
- `serve_itinerary.js` - local Node.js viewer for the itinerary, map data, resource links, and live river gauge references.

## Run Locally

This project uses only Node.js built-in modules.

```sh
node serve_itinerary.js
```

Then open:

```text
http://localhost:8123
```

## Notes

River days are flow-dependent. Re-check American Whitewater, CDEC, USGS, and local conditions before committing to any run.

Dispersed camping notes are planning-level only. Confirm seasonal closures, MVUM access, fire restrictions, and posted stay limits before camping.
