Import option

For a visual map with pins, import trip_stops.csv into Google My Maps:
- https://www.google.com/mymaps
- Create a new map
- Import trip_stops.csv as a layer
- Use the date or type column for styling if you want to color-code friend stays, van camps, river days, hot springs, and backpacking blocks
